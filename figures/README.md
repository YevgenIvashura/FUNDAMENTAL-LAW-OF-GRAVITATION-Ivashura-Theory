# Figures

This folder contains figures, diagrams, and screenshots related to the Ivashura Theory and its experimental validation.

## Recommended structure

- `figures/experimental/` — setup screenshots, prototype photos
- `figures/diagrams/` — theory diagrams, conceptual схемы
- `figures/plots/` — graphs and plots

## Notes

Use descriptive filenames, e.g.:
- `setup_v1_photo_2026-01.png`
- `density_dependence_plot.png`
